# ThermoCoupleLog
Firmware for ThermoCoupler




High Speed 8 Channel Temperature measurement:

**[PC]** - USB - **[MCP2200]** - UART - **[PIC16F1512]** - SPI - **[MAX31855]** - Analog - **[Thermocouple]**

**Parameters:**

- Up to 200 Scans/second [8-Channels in parallel]
